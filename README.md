# 8阶光立方模拟器（轻量版）

## 在线模拟网站  
<a href="https://everygg.com">8阶光立方模拟器</a>

## 简介
可导出动画数据  
不包含后端程序，可直接运行    

## 动画数据格式
每次65个数据为一组（一个场景）其中1个为时间，64个为led状态。  
需要详细说明请联系作者  

## 使用说明
* 片操作  
片选代表哪个平面，如XY则代表XY平面。  
片层范围为0-7，分别代表某平面上的0-7层  
* 环操作  
需结合片选和片亮。  
环阶代表某平面上中心点从里往外正方形的边长  
环层范围为0-7，分别代表某平面上的0-7层  
* 块操作  
块阶代表立方体的边长  
块坐标代表立方体最左下角在光立方上的坐标，以此定位块立方体的位置

## 名称
* 场景（场景画面数据）  
一次512个led亮灭状态。一次时间固定为0.1s。（其中每片亮10μs，10μs * 8 + 20μs）
* 场景持续时间  
最小单位为0.1s。例如1则代表0.1s，2则代表0.2s
* 动画  
由很多场景+场景持续时间构成
* 动画数据  
采用16进制来表述动画。将场景+场景持续时间用数据表示。  
每次65个数据为一组（一个场景）其中1个为时间，64个为led状态。  
* 片  
从下往上，依次为0-7片
* 行  
某片中，从里往外依次为0-7行
* 坐标系  
按照右手坐标。
x：从左往右，y：从下往上，z：从里往外
